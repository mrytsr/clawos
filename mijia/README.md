## 下一步

1. **打开Excel文件** 查看设备列表
2. **从米家APP获取Token**：
   - 打开米家APP → 我的 → 设置 → 设备信息
   - 查看每台设备的Token
   - 填入Excel表格
3. **控制设备**：
   ```bash
   miiocli miotdevice --ip <IP> --token <TOKEN> status
   ```
